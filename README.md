manual build

how to build
```bash 
docker-compose up --build
```

how to stop 
```bash
docker-compose down
```

forse delete
```bash
docker-compose down --rmi all
```

address in web
```bash
http://localhost:3000
```

automatic build 
```bash
# build
./install -b

# restart
./install -r

# delete all
./install -d 

# stop
./install -s
```
