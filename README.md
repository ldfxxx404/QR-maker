### Manual

```bash
# Build & start
docker-compose up --build

# Stop
docker-compose down

# Force remove images
docker-compose down --rmi all
```

[localhost:3000](http://localhost:3000)

---

### Automatic (`./install`)

```bash
-b    Build
-r    Restart
-s    Stop
-d    Delete all
-dev  npm run dev
```
